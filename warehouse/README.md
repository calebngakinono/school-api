# Warehouse

This is the start of a project that will help manage a warehouse. It only has a single class right now, but it'll be expanded later.

## Getting started

1. Rename `.env.example` to `.env`.
1. Create virtual environment
1. Activate virtual environment
1. Install dependencies

## Requirements

Create a FastAPI app that includes all the basic CRUD operations for the `Item` class. Do _not_ use AI.
